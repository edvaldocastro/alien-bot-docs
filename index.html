Esta documentacao foi preparada para administradores que querem configurar e usar o bot em seus grupos de WhatsApp. Cada comando abaixo usa o formato {prefixo}comando, porque o prefixo pode mudar de grupo para grupo.

Inicio rapido

Primeiros passos para usar no grupo

Use esta sequencia ao adicionar o bot em um grupo novo.

1. Confirme se o bot esta respondendo

Use este comando para validar que o bot esta online no grupo.

{prefixo}ping

2. Abra o menu administrativo

Administradores usam este menu para ver protecoes, configuracoes e acoes de moderacao.

{prefixo}menuadm

3. Defina o prefixo do grupo

Troca o prefixo dos comandos. Depois disso, comandos como {prefixo}menu passam a usar o novo prefixo escolhido.

{prefixo}setprefix .

4. Configure regras e protecoes essenciais

Cadastre regras do grupo ou defina as regras na descricao do grupo, pois o bot busca o texto de la para exibir. Ajuste tambem protecoes como antilink, antispam, antifake e DDIs permitidos.

{prefixo}setrules <texto>

5. Confira o estado final

Mostra protecoes ativas, configuracoes do grupo e controles globais relevantes.

{prefixo}status
